## 中心差分公式详解

### 1. 导数的本质

导数的定义是：

$$f'(x_0) = \lim_{h \to 0} \frac{f(x_0 + h) - f(x_0)}{h}$$

但计算机**无法让 $h$ 真正趋近于 0**，只能取一个很小的数（如 $h = 10^{-6}$）。这就产生了**数值导数**。

---

### 2. 三种数值求导方法

#### 方法一：前向差分（Forward Difference）

$$f'(x_0) \approx \frac{f(x_0 + h) - f(x_0)}{h}$$

只用 $x_0$ 右边一个点。

#### 方法二：后向差分（Backward Difference）

$$f'(x_0) \approx \frac{f(x_0) - f(x_0 - h)}{h}$$

只用 $x_0$ 左边一个点。

#### 方法三：中心差分（Central Difference）——最常用

$$f'(x_0) \approx \frac{f(x_0 + h) - f(x_0 - h)}{2h}$$

**同时用左右两个点**，取平均。

---

### 3. 为什么中心差分更好？（精度对比）

用泰勒展开来分析误差：

$$f(x_0 + h) = f(x_0) + h \cdot f'(x_0) + \frac{h^2}{2} f''(x_0) + \frac{h^3}{6} f'''(x_0) + O(h^4)$$

$$f(x_0 - h) = f(x_0) - h \cdot f'(x_0) + \frac{h^2}{2} f''(x_0) - \frac{h^3}{6} f'''(x_0) + O(h^4)$$

**前向差分：**

$$\frac{f(x_0+h) - f(x_0)}{h} = f'(x_0) + \frac{h}{2}f''(x_0) + O(h^2)$$

误差项是 $\frac{h}{2}f''(x_0)$，即 **$O(h)$** —— 精度较低。

**中心差分：**

两式相减：

$$f(x_0+h) - f(x_0-h) = 2h \cdot f'(x_0) + \frac{h^3}{3}f'''(x_0) + O(h^4)$$

除以 $2h$：

$$\frac{f(x_0+h) - f(x_0-h)}{2h} = f'(x_0) + \frac{h^2}{6}f'''(x_0) + O(h^3)$$

误差项是 $\frac{h^2}{6}f'''(x_0)$，即 **$O(h^2)$** —— 精度更高！

---

### 4. 直观理解：为什么"左右两边取平均"更准？

```
        f(x)
         |
         |        /  真实曲线
         |      /
    B ---●----/-------- f(x0+h)
         |  / |
         | /  |
   A ----●----|-------- f(x0-h)
         |    |
         |    |
         +----+---------- x
              x0

前向差分: 用 A 和 B 的连线斜率 → 偏大
后向差分: 用 A 和 C 的连线斜率 → 偏小
中心差分: 用 C 和 B 的连线斜率 → 恰好穿过切线点，最接近真实导数
```

前向/后向差分只看一侧，容易受曲线弯曲方向的影响而产生系统偏差。中心差分**两侧对称取点**，弯曲误差互相抵消了一阶项，只剩下二阶项。

---

### 5. 精度 $O(h^2)$ 是什么意思？

| 方法 | 误差 | $h = 0.01$ 时的误差量级 | $h = 0.001$ 时的误差量级 |
|---|---|---|---|
| 前向差分 | $O(h)$ | ~0.01 | ~0.001 |
| 中心差分 | $O(h^2)$ | ~0.0001 | ~0.000001 |

同样取 $h = 0.001$，中心差分的误差比前向差分小 **1000 倍**。

---

### 6. 对应代码

```python
def numerical_derivative(func, x0, args=(), dx=1e-6):
    return (func(x0 + dx, *args) - func(x0 - dx, *args)) / (2 * dx)
    #     ~~~~~~~~~~~~~~~~          ~~~~~~~~~~~~~~~~~       ~~~~
    #       f(x0 + h)                f(x0 - h)             2h
```

这就是中心差分公式的直接翻译，`dx` 就是公式中的 $h$。
